============================================================
  Clone Hero Chart Manager  (CHM)
============================================================

WHAT IT IS
----------
A small Windows app for searching, downloading and installing Clone Hero
charts from RhythmVerse and Chorus Encore. RockBand Xbox360 files
(CON / .rb3con) are automatically converted into a format Clone Hero
can read. Drag-and-drop installs are supported for files you downloaded
manually from MEGA, Mediafire or anywhere else.


INSTALL (easy)
--------------
Just run the included "CHM-Setup-<version>.exe" installer.

Everything the app needs (Onyx for CON conversion, modern 7-Zip for
archives, .sng unpacker) is bundled inside, so there are no extra
downloads. The installer creates Start-menu and desktop shortcuts.

On first launch, if your Clone Hero "Songs" folder isn't detected
automatically, the app opens Settings — point it at your "Songs"
folder once and you're done.


PORTABLE / FOLDER VARIANT
-------------------------
If you're using the zipped folder bundle instead of the installer:

1) Copy this WHOLE folder somewhere on your PC.
2) Run "Clone Hero Chart Manager.exe" inside it.

   Note: keep the "onyx" and "tools" folders next to the .exe —
   without them conversion and extraction won't work.


CONTROLS
--------
  /  ............. jump to search
  arrows ......... select a song
  Enter .......... download the selected song
  PageUp/Down .... next / previous page
  Esc ............ hide the window

  Title bar:
    🎮 Launch / Switch to Clone Hero   (lights up green while CH is running)
    📁 Library manager
    ⚙  Settings
    ▁  Hide to system tray
    ✕  Quit

  Optional: in Settings you can bind a global "Show / hide" hotkey if
  you want to bring the app up from anywhere — most users just use
  Alt+Tab.


HOW DOWNLOADED SONGS SHOW UP IN THE GAME
----------------------------------------
Clone Hero cannot refresh its library from outside. After downloading,
click "Launch / Switch to Clone Hero" in the title bar and pick:
  Settings → General → Scan Songs.
Your new songs will then appear.


IF THE "Songs" FOLDER ISN'T FOUND
---------------------------------
Open Settings (⚙) in the app and set the path to your Clone Hero
"Songs" folder manually.


Have fun!  🎸
